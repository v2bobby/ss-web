# SomaSync

> Precision anatomy. Public health impact.

**SomaSync** is a high-end health literacy platform that bridges human anatomy and public health. It demystifies complex medical conditions by rooting them in fundamental mechanistic understanding — not just symptoms.

Designed for global reach, low-bandwidth environments, and measurable scholarship impact.

---

## ✨ Highlights

- **Biology-First Content Architecture** — Every article starts with normal anatomy before pathology.
- **Interactive Body Map** — Visual discovery engine optimized for low-literacy and high-stress users.
- **Myth-Busters** — Clinical, anatomical debunking of common health misinformation.
- **Knowledge Check System** — Built-in 3-question quizzes with real literacy-lift analytics.
- **Community Insights Dashboard** — Heatmap of anatomical concerns for public health targeting.
- **Performance-First** — SVG diagrams, static generation, aggressive caching. Works on 3G.

---

## 🛠 Tech Stack

- **Frontend**: Vanilla HTML + Tailwind CSS (via CDN) + Chart.js
- **Interactivity**: Pure JavaScript (modals, quiz engine, body map, dynamic chart)
- **Deployment**: Vercel (static)
- **Design**: Medical Blue (#0A66C2) + Zinc dark theme, Space Grotesk + Inter typography
- **Assets**: Custom AI-generated hero visual

No build step required. Pure static site.

## 🚀 Quick Start

### Local Development

```bash
# Clone or download the project
cd somasync-website

# Option 1: Simple Python server
python3 -m http.server 3000

# Option 2: npx serve (if Node is available)
npx serve .
```

Open [http://localhost:3000](http://localhost:3000)

### One-Click Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-org/somasync)

Or manually:
1. Push this folder to GitHub
2. Import project in Vercel
3. Deploy (zero configuration needed)
